# Streamlit Demo — README

Kurze Anleitung, um die `streamlit_demo.py` lokal auszuführen.

1. (Optional) Erstelle ein virtuelles Environment:

```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

2. Abhängigkeiten installieren:

```bash
pip install -r requirements.txt
```

3. Streamlit App starten:

```bash
streamlit run streamlit_demo.py
```

Die App zeigt Beispiele für Widgets, Layout, Charts (Altair/Plotly), Map, Dateiupload und Media.

Wenn du bestimmte Beispiele (z. B. Audio-Playback, Video) ergänzt haben möchtest, sag mir kurz, welche Medien oder Daten du testen willst.
